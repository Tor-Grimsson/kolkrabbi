# Kolkrabbi Monorepo

## Project Structure
- `apps/web` - Main website (React + Vite + Tailwind)
- `apps/studio` - Sanity CMS
- `packages/fontviewer` - Reusable font viewer package

## Commands
- `npm run dev:web` - Run website dev server
- `npm run dev:studio` - Run Sanity Studio
- `npm run build:web` - Build website for production
- `npm install` - Install all dependencies for all workspaces
